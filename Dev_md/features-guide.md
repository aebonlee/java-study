## 1.5. Java 코드 실행기 (JavaCodeRunner)

### 기능
- **코드 편집**: react-simple-code-editor + PrismJS 구문 강조
- **실행**: Piston API (https://emkc.org/api/v2/piston/execute) - 무료, 인증 불필요
- **출력**: 실행 결과 또는 에러 메시지 표시
- **초기화**: 기본 코드로 리셋

### Props
| Prop | 타입 | 설명 |
|------|------|------|
| `defaultCode` | string | 기본 Java 코드 |
| `title` | string | 실행기 제목 |
| `readOnly` | boolean | 읽기 전용 모드 |

### 사용 예시
```jsx
<JavaCodeRunner
  defaultCode={`public class Main {
    public static void main(String[] args) {
        System.out.println("Hello!");
    }
}`}
  title="Hello World 실행"
/>
```

## 2. 퀴즈 센터 (`/quiz`)

### 퀴즈 목록
| ID | 제목 | 문제 수 | 제한시간 | 합격점 |
|----|------|---------|---------|--------|
| basics | Java 기초 퀴즈 | 10 | 10분 | 70점 |
| intermediate | Java 중급 퀴즈 (OOP) | 10 | 10분 | 70점 |
| advanced | Java 고급 퀴즈 | 10 | 12분 | 70점 |
| web | 웹 개발 퀴즈 | 10 | 12분 | 70점 |

### 퀴즈 기능
- **보기 셔플**: 매번 보기 순서가 랜덤으로 변경
- **타이머**: 제한시간 경과 시 자동 제출 (60초 미만 경고 애니메이션)
- **정답/오답 표시**: 정답은 초록색, 오답은 빨간색
- **해설**: 각 문제별 해설 제공
- **결과 화면**: 점수, 합격/불합격, 문제별 정답 여부
- **다시 풀기**: 무제한 재시도 가능
- **퀴즈 이력**: 최근 3회 시도, 최초/최근 응시일, 합격 상태

### 데이터 구조 (quizzes.js)
```javascript
{
  title: '퀴즈 제목',
  description: '설명',
  passingScore: 70,
  timeLimit: 600, // 초 단위
  questions: [
    {
      question: '문제 텍스트',
      options: ['선택지1', '선택지2', '선택지3', '선택지4'],
      correct: 0, // 정답 인덱스
      explanation: '해설'
    }
  ]
}
```

## 3. 도장깨기 - 배지 시스템 (`/badges`)

### 배지 등급 (총 24개)
| 등급 | 개수 | 색상 |
|------|------|------|
| 브론즈 | 6 | #CD7F32 |
| 실버 | 7 | #C0C0C0 |
| 골드 | 8 | #FFD700 |
| 플래티넘 | 4 | #a78bfa |

### 배지 조건 유형
| 유형 | 설명 | 예시 |
|------|------|------|
| `lessons_completed` | N개 레슨 완료 | 5개 레슨 완료 |
| `level_completed` | 특정 단계 완료 | 기초 단계 완료 |
| `quiz_passed` | 특정 퀴즈 N점 이상 | 기초 퀴즈 70점 이상 |
| `quiz_perfect` | 특정 퀴즈 만점 | 기초 퀴즈 100점 |
| `all_quizzes_passed` | 모든 퀴즈 통과 | 4개 퀴즈 모두 70점+ |
| `all_quizzes_perfect` | 모든 퀴즈 만점 | 4개 퀴즈 모두 100점 |
| `code_runs` | 코드 N회 실행 | 50회 이상 실행 |
| `all_completed` | 모든 레슨 완료 | 43개 레슨 모두 |
| `specific_lessons` | 특정 레슨 완료 | Ch.15 Spring 완료 |
| `java_master` | 전체 레슨 + 퀴즈 | 모든 것 완료 |

### 배지 획득 알림
- 조건 달성 시 화면 중앙에 팝업 표시
- 배지 아이콘, 이름, 설명 표시
- 클릭하면 팝업 닫힘

### 배지 저장
- localStorage(`javamaster-badges`)에 획득한 배지 ID 배열 저장
- BadgeContext에서 실시간 조건 평가

## 7. 로그인 시스템 (`/login`)

### OAuth 로그인
- **Google 로그인**: Supabase Auth + Google OAuth
- **Kakao 로그인**: Supabase Auth + Kakao OAuth
- Supabase 미설정 시 에러 메시지 표시

### 세션 관리
- 30분 자동 세션 만료
- 만료 5분 전 경고 배너 표시
- "연장하기" 버튼으로 세션 갱신
- localStorage(`javamaster-session-expiry`)에 만료 시간 저장

### Navbar 사용자 메뉴
- **로그인 전**: "로그인" 버튼 표시
- **로그인 후**: 사용자 아바타 표시
  - 클릭 시 풍선 드롭다운 메뉴
  - 사용자 이름, 이메일
  - 마이페이지 링크
  - 관리자 메뉴 (관리자 이메일만)
  - 로그아웃 버튼

### 로그인 모달 (requireAuth)
- 로그인 필요 기능 접근 시 자동 팝업
- Google/Kakao 로그인 버튼
- 로그인 완료 후 원래 작업 자동 수행
