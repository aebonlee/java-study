## 1.5. Java 코드 실행기 (JavaCodeRunner)

### 기능
- **코드 편집**: react-simple-code-editor + PrismJS 구문 강조
- **실행**: Piston API (https://emkc.org/api/v2/piston/execute) - 무료, 인증 불필요
- **출력**: 실행 결과 또는 에러 메시지 표시
- **복사**: 제목 옆 복사 버튼으로 코드 클립보드 복사 (2초간 "복사됨" 피드백)
- **초기화**: 기본 코드로 리셋

### 코드 블록 복사 (CodeBlockCopy)
- 모든 레슨의 `.code-block` 헤더 우측에 복사 버튼 자동 삽입
- `CodeBlockCopy.jsx` 전역 컴포넌트가 페이지 이동 시 자동 감지
- `navigator.clipboard.writeText()`로 코드 복사
- 클릭 시 2초간 "복사됨" + 초록색 체크 피드백
- JavaCodeRunner(실행기)와 별개로 모든 정적 코드 블록에 적용

### Props
| Prop | 타입 | 설명 |
|------|------|------|
| `defaultCode` | string | 기본 Java 코드 |
| `title` | string | 실행기 제목 |
| `readOnly` | boolean | 읽기 전용 모드 |

### 사용 예시
```jsx
<JavaCodeRunner
  defaultCode={`public class Main {
    public static void main(String[] args) {
        System.out.println("Hello!");
    }
}`}
  title="Hello World 실행"
/>
```

## 2. 퀴즈 센터 (`/quiz`)

### 퀴즈 목록
| ID | 제목 | 문제 수 | 제한시간 | 합격점 |
|----|------|---------|---------|--------|
| basics | Java 기초 퀴즈 | 10 | 10분 | 70점 |
| intermediate | Java 중급 퀴즈 (OOP) | 10 | 10분 | 70점 |
| advanced | Java 고급 퀴즈 | 10 | 12분 | 70점 |
| web | 웹 개발 퀴즈 | 10 | 12분 | 70점 |
| servlet | 서블릿 퀴즈 | 10 | 12분 | 70점 |
| spring | Spring 퀴즈 | 10 | 12분 | 70점 |

### 퀴즈 기능
- **보기 셔플**: 매번 보기 순서가 랜덤으로 변경
- **타이머**: 제한시간 경과 시 자동 제출 (60초 미만 경고 애니메이션)
- **정답/오답 표시**: 정답은 초록색, 오답은 빨간색
- **해설**: 각 문제별 해설 제공
- **결과 화면**: 점수, 합격/불합격, 문제별 정답 여부
- **다시 풀기**: 무제한 재시도 가능
- **퀴즈 이력**: 최근 3회 시도, 최초/최근 응시일, 합격 상태

### 데이터 구조 (quizzes.js)
```javascript
{
  title: '퀴즈 제목',
  description: '설명',
  passingScore: 70,
  timeLimit: 600, // 초 단위
  questions: [
    {
      question: '문제 텍스트',
      options: ['선택지1', '선택지2', '선택지3', '선택지4'],
      correct: 0, // 정답 인덱스
      explanation: '해설'
    }
  ]
}
```

## 7. 로그인 시스템 (`/login`)

### 로그인 방식
- **이메일 로그인**: Supabase Auth signInWithPassword (v3.4.0)
- **이메일 회원가입**: Supabase Auth signUp + 이메일 확인 (v3.4.0)
- **Google 로그인**: Supabase Auth + Google OAuth
- **Kakao 로그인**: Supabase Auth + Kakao OAuth
- Supabase 미설정 시 에러 메시지 표시

### 세션 관리
- 30분 자동 세션 만료
- 만료 5분 전 경고 배너 표시
- "연장하기" 버튼으로 세션 갱신
- localStorage(`javamaster-session-expiry`)에 만료 시간 저장

### Navbar 사용자 메뉴
- **로그인 전**: "로그인" 버튼 표시
- **로그인 후**: 사용자 아바타 표시
  - 클릭 시 풍선 드롭다운 메뉴
  - 사용자 이름, 이메일
  - 마이페이지 링크
  - 관리자 메뉴 (관리자 이메일만)
  - 로그아웃 버튼

### 로그인 모달 (requireAuth)
- 로그인 필요 기능 접근 시 자동 팝업
- 로그인/회원가입 탭 전환
- 이메일/비밀번호 입력 폼
- "또는" 구분선 아래 Google/Kakao OAuth 버튼
- 로그인 완료 후 원래 작업 자동 수행
